Core Components

API Gateway

Central entry point for all client requests. Handles authentication, rate limiting, and request routing.

Kong Nginx

Auth Service

Manages user authentication, authorization, and session management using JWT tokens.

Node.js PostgreSQL

Product Service

Handles product catalogue, inventory management, and search functionality.

Python MongoDB

Order Service

Processes orders, manages cart operations, and coordinates with payment service.

Go PostgreSQL

Payment Service

Integrates with payment gateways and handles transaction processing.

Java Stripe API

Notification Service

Sends email, SMS, and push notifications to users based on events.

Node.js SendGrid
